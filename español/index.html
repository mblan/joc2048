Vino + Matemáticas = JOC 2048 Juega!
Junta los números iguales hasta conseguir 2048.

Vuelve a jugar

Reglas de juego: Utiliza las flechas del teclado para mover todas las filas o columnas. Cada vez que se junten dos números iguales, se irán sumando: 2, 4, 8, 16... El objectivo es llegar a 2048!

*Si utilizas un dispositivo con pantalla táctil, mueve las filas y columnas con el dedo.

ATENCIÓN: Juega siempre con moderación !!!
